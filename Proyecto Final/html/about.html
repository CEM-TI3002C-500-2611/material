Sobre Nosotros

Somos un grupo de alumnos del Tecnológico de Monterrey que estamos desarrollando un proyecto de analítica predictiva para Liverpool.

Proyecto

Colaboración académica para Liverpool

Este proyecto integra análisis descriptivo y modelos de predicción para apoyar la toma de decisiones en la gestión de marcas dentro de Liverpool.

Nuestro objetivo es transformar datos en información accionable para estimar riesgo comercial y visualizar escenarios de venta a futuro.

Equipo

Alumnos del Tecnológico de Monterrey

Somos un equipo multidisciplinario de estudiantes que combina habilidades de ciencia de datos, desarrollo web e ingeniería de software para construir soluciones aplicadas al retail.

Tecnologías

Stack utilizado en el proyecto

Tableau

Para construir y publicar tableros descriptivos e interactivos.

FastAPI

Para el desarrollo del backend y la exposición de endpoints de predicción.

Dash

Para el frontend analítico y la visualización de resultados.

Scikit-Learn

Para entrenar y evaluar el modelo predictivo de riesgo comercial.
